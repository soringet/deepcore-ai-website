# AI Deep Core Research — Static Website

This is a clean, enterprise-grade static website you can deploy anywhere (Hostinger, Netlify, Vercel static, GitHub Pages, S3, etc.).

## Structure
- `index.html` — homepage
- `pages/` — site pages
- `assets/css/styles.css` — styling
- `assets/js/main.js` — mobile menu + simple form message

## Run locally
Option A (Python):
```bash
python -m http.server 8000
```
Then open: http://localhost:8000

Option B (Node):
```bash
npx serve .
```

## Contact form
The form is static. Connect it to:
- Formspree, Basin, Netlify Forms
- Or your own backend endpoint / CRM
